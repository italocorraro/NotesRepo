Appunti sulla Sintassi di html

Questa pagina copre le basi della sintassi html e gli attributi.

Overview

HTML è un linguaggio di markup, cioè struttura la pagina web dal punto di vista principalmente testuale.

HTML NON è un linguaggio di programmazione, in quanto non esegue nessuna istruzione né calcolo; viene semplicemente interpretato dal browser al momento del render della pagina (non viene compilato).

Sintassi base

html consiste di una serie di elementi racchiusi da tag:

  • Il tag di apertura indica l'inizio dell'elemento ed è dove vengono elencati i suoi attributi.
  • Nel mezzo sta' il contenuto, che è sempre testo.
  • Il tag di chiusura è lo stesso dell'apertura preceduto da /.

il browser non restituisce errore in mancanza di un tag di chiusura o nel caso di una chiusura incrociata; sono comunque entrambe pratiche da evitare.

I tag html possono essere (naturalmente) innestati.

Elementi Void

Alcuni tag non richiedono un tag di chiusura, questi sono chiamati elementi void

sebbene non richiesto, è sempre bene chiudere gli elementi void con un / alla fine del tag di apertura (unico) prima di >.

Commenti

I commenti in html si racchiudono tra "<!--"" e "-->".

Entità

Simboli e caratteri riservati a html devono essere sostituiti da entità per non essere scambiati per tag dal browser.

un'entità è preceduta da & e termina per ;
le entità possono essere chiamate poi per nome oppure con un codice numerico; in quest'ultimo caso, il numero deve essere preceduto da #

Struttura

Un documento html ha sempre questa struttura:

La prima riga è la dichiarazione del tipo di documento
il primo tag che deve contenere tutto è html

l'elemento head contiene i metadati del documento; il suo contenuto non è visualizzato dal browser, ma viene utilizzato come informazioni sulla pagina

l'elemento body contiene tutta la parte visualizzata di "codice"
