Appunti sulle tabelle in html

Questa pagina copre l'argomento delle tabelle in html.

Overview

Le tabelle in html sono definite tramite l'elemento table;

servono a strutturare contenuto in una griglia composta da righe e colonne.

Le tabelle venivano usate per generare layout, ma oggi ci sono metodi migliori.

Elementi Base

Le tabelle in html sono naturalmente composte di righe, a loro volta divise in celle che rispettano le dimensioni della colonna a cui appartengono

Riga di Tabella

Una riga è definita tramite l'elemento tr (table row);

una cella tramite l'elemento td (table data)

dato1.1 dato1.2 dato1.3
dato2.1 dato2.2 dato2.3

Si possono definire anche delle celle header con th (table header)

H1 H2 H3
dato1.1 dato1.2 dato1.3
dato2.1 dato2.2 dato2.3

Celle Unite

Per permettere a una cella di spaziare più di una singola riga e colonna si usano gli attributi colspan e rowspan

Elementi di Struttura

Una tabella grande risulta difficile da leggere; occorre fornire alcuni elementi strutturali:

  • thead: include la parte di header della tabella
  • tbody: include il corpo della tabella
  • tfoot: include il termine della tabella

tbody è sempre implicitamente inclusa in una tabella, anche quando non specificata

Attributo scope

Per indicare l'appartenenza di un gruppo di celle ad un certo header possiamo usare l'attributo scope, che accetta uno di quattro valori:

  • col per l'header di una colonna
  • row per l'header di una riga
  • colgroup per un header che spazia su più colonne
  • rowgroup per un header che spazia su più righe
