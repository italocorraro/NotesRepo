Overview

In html si possono definire tre diversi tipi di liste.

Liste Generiche

Possiamo definire una semplice lista per la quale l'ordine degli elementi non è importante con il tag ul (unordered list).

Ogni oggetto della lista deve essere racchiuso nel tag li (list item)

Il risultato è

  • elemento-1
  • elemento-3
  • elemento-2

Liste Innestate

Possiamo anche creare una lista all'interno di un'altra inserendola all'interno di uno dei suoi elementi.

  • elemento-1
  • elemento-2
    • elemento-2.1
    • elemento-2.2
  • elemento-3

Liste Ordinate

Possiamo definire una lista di elementi ordinati con il tag ol (ordered list).

  1. oggetto-1
  2. oggetto-2
    1. oggetto-2-1
    2. oggetto-2-2
  3. oggetto-3
    • oggetto-2.1
    • oggetto-2.2

Non è necessario che le liste siano dello stesso tipo per poterle innestare.

Liste Informative

Le liste informative usano sono racchiuse dal tag dl (description list);

ogni elemento della lista dal tag dt (description term)

e le relative informazioni nel tag dd (description definition)

term-1
shmuck-1
