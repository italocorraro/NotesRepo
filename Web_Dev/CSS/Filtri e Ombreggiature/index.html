Overview

Ombre

La proprietà box-shadow applica effetti di ombreggiatura attorno alla cornice di un elemento.

L'unica keyword che questa proprietà accetta è none, che serve a disabilitarla.

La proprietà box-shadow accetta fino a 6 valori:

  • i primi quattro numeri vengono interpretati come segue
    • i primi due valori numerici indicano l'offset lungo gli assi dell'ombra rispetto all'elemento (da tenere a mente che le direzioni positive sono verso destra per l'asse orizzontale e verso il basso per l'asse verticale);

      se uno o entrambi i valori mancano, vengono impostati di default a 0

    • il terzo valore (opzionale; default a 0) numerico viene interpretato come raggio di sfocatura dell'ombra (dev'essere positivo)
    • il quarto valore (opzionale) numerico viene interpretato come il raggio di espansione dell'ombra (default a 0)
  • opzionalmente si può specificare un colore (anche con funzioni rgb o hsl) per l'ombra; se non specificato viene usato il valore della proprietà color dell'elemento
  • opzionalmente si può aggiungere il valore inset;

    di default l'ombra è proiettata verso l'esterno dell'elemento, ma con questo valore verrà proiettata verso l'interno
