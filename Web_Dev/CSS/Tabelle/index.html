Overview

Spaziatura tra Contorni

Possiamo impostare la distanza tra i contorni delle celle adiacenti con la proprietà border-spacing;

questa proprietà accetta fino a due valori di lunghezze, uno per la distanza orizzontale, l'altro per quella verticale; se viene immesso un solo valore allora viene usato per entrambe le direzioni.
