Appunti sulle media queries in CSS

Questa pagina copre l'argomento della responsività delle pagine web con Media Queries in CSS.

Overview

Le Media Query permettono di applicare regole di CSS selettivamente in base a delle regole sulle caratteristiche della viewport.

Media Query

La sintassi di base per le media queries è

  • il tipo di media si riferisce o a uno schermo (con un browser) (screen) oppure a un dispositivo tipo stampante (per stampare il documento) (print)
  • un'espressione (relativa alla viewport) che deve essere rispettata perché il contenuto della query venga applicato
  • un set di regole CSS da utilizzare se l'espressione è corretta

si possono impostare più regole per la stessa query separandole con un connettore logico and, una virgola o not

Media Rules

Le regole più utili da impostare per le Media Queries:

Altezza e Larghezza

Applicare selettivamente delle regole CSS in base alle dimensioni della viewport

Regola Descrizione Esempio
max-height il CSS è applicato solo se la viewport ha altezza minore o uguale al valore specificato
min-height il CSS è applicato solo se la viewport ha altezza maggiore o uguale al valore specificato
height il CSS è applicato solo se la viewport ha altezza uguale al valore specificato
max-width il CSS è applicato solo se la viewport ha larghezza minore o uguale al valore specificato
min-width il CSS è applicato solo se la viewport ha larghezza maggiore o uguale al valore specificato
width il CSS è applicato solo se la viewport ha larghezza uguale al valore specificato
aspect-ratio il CSS è applicato solo se la viewport rapporto larghezza / altezza uguale al valore specificato

le misure della viewport includono anche le scrollbar

Orientamento Schermo

La regola può anche basarsi sull'orientamento schermo rilevato, cioè alle dimensioni relative di altezza e larghezza della viewport

Regola Descrizione Esempio
portrait il CSS è applicato solo se la viewport ha altezza maggiore della larghezza (il dispositivo è lungo)
landscape il CSS è applicato solo se la viewport ha larghezza maggiore dell'altezza (il dispositivo è largo)
