Appunti sul Box Model in CSS

Questa pagina copre l'argomento della Box Model.

Overview

La box model è la rappresentazione di ogni elemento in html e consiste di quattro parti: contenuto, padding, contorno e margine.

Box Model

Nel box model, il contenuto dell'elemento è mostrato in un contenitore rettangolare, attorno a questo rettangolo c'è dello spazio (padding) che lo separa dal contorno, infine oltre il contorno c'è dello spazio (margine) a separarlo dal contenitore superiore.

Il margine è sempre trasparente.

l'altezza e la larghezza che vengono impostate come proprietà di un elemento sono riferite SOLO al contenuto!
Per conoscere l'altezza e larghezza effettive di un elemento bisogna aggiungere il padding e il contorno:

calcoliamo:
larghezza effettiva =
+320px (width)
+20px (padding-left + padding-right)
+10px (border-left + border-right)
= 350px (larghezza effettiva) analogamente per l'altezza:
altezza effettiva =
+50px (height)
+20px (padding-top + padding-bottom)
+10px (border-top + border-bottom)
= 80px (altezza effettiva)

La proprietà box-sizing

La proprità box-sizing può avere questi valori:

  • content-box: le proprità di altezza e larghezza includono solo il contenuto.

    questo è il valore di default per ogni elemento.

  • border-box: le proprietà di altezza e larghezza includono contenuto, padding e contorno.

se impostiamo avremo la situazione opposta a quella di default: le dimensioni del contenuto saranno pari a quelle dell'elemento tolti padding e contorno.
